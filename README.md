# AllTestDemo

A platform where users can do basic operations such as adding, deleting, modifying, logging in, and logging out.

## Clone

```
git clone https://github.com/johnny85022616/allTestDemo.git
```

## Quick Overview
```
npm install 
npm run build 
npm run start:production_server
```
